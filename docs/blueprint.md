# **App Name**: EduGestion

## Core Features:

- User Authentication: Secure login for administrators and guests with role-based access control.
- Student Data Management: Digitalize student records including personal details (name, ID, contact info), academic history (enrollment year, grades, attendance), and responsible adult contacts.
- Teacher Data Management: Maintain teacher profiles with personal data, contact information, qualifications, and course assignments (schedule, subject).
- Admin Data Control: Authorized personnel can add, modify, or remove student and teacher data.
- Guest Regularity Update: Preceptors update student's attendance/regularity status.

## Style Guidelines:

- Primary color: A serene blue (#64B5F6) to convey trust and stability.
- Background color: Light blue (#E3F2FD), creating a calm and professional environment.
- Accent color: Yellow (#FFCA28) to highlight important actions and notifications.
- Body and headline font: 'PT Sans' for a modern, readable, humanist sans-serif that is easy to read in any format.
- Use clear, professional icons for navigation and data representation.
- Implement a clean, structured layout for efficient data management and navigation.
- Use subtle transitions to improve user interaction without being distracting.
